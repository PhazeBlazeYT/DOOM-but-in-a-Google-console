# DOOM-but-in-a-Google-console

Paste the contents of `doom-console.js` into any page's browser console.

What this version includes:
- Draggable/resizable game window
- Raycasted walls with depth shading
- Enemy sprites with simple pursuit AI
- Shooting with line-of-sight hit checks
- Health/ammo/kill HUD
- Win state (kill all enemies + reach exit tile)

> Note: this is still a lightweight JavaScript homage, not the original DOOM engine.
